# 그라운드 플립 발표자료·기획서 (SW마에스트로 15기 M3 PRO)

> [!tldr]
> 선배 팀 M3 PRO(차상민·김민욱·구민)의 "그라운드 플립"(지도 기반 게이미피케이션 걷기 앱) 최종 발표자료(pptx)와 SW마에스트로 15기 프로젝트 기획서(pdf).
> 성과 지표: 누적 사용자 2,640명 · 방문한 땅 81,027개 · 앱스토어 평점 4.7/구글 4.5 · velog 트렌딩 상위 · 협찬/기사 다수 — FillMap이 벤치마킹할 성과 서사 구조.
> 기획서는 소마 기획서 양식의 실제 합격 사례 — 목적·개요·수행방안·일정 구성 참고.

## 요약
- **최종 발표자료 (pptx, 슬라이드 ~18장)**: 프로젝트 소개 → 주요 성과(방문 땅 81,027개·1.5억 평, 서울 85.7% 커버, 누적 사용자 2,640명, GA 지표·스토어 순위·평점 4.5/4.7) → 협찬·홍보(모어케이션·캐시워크·렛플)·언론 기사 → velog 트렌딩·인기작가 → 향후 계획·BM.
- **기획서 (pdf)**: 2024년 SW마에스트로 15기. 걷기 습관화 게이미피케이션. Flutter + Spring Boot 하이브리드 앱. 일정: 기획·설계(6월) → 개발(6~8월) → 프로토타입 배포(8월) → 배포·개선(9~11월).
- FillMap과의 관계: 격자 점령·지도 도메인이 거의 동일 — 기술 사례는 [[그라운드 플립 아티클 모음]] 하위 노트 참고.
